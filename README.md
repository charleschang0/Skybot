# SKYBOT Static Website

Static website draft for `skybot.korvva.com`.

## Files

- `index.html` - single-page static site
- `styles.css` - responsive layout and visual system
- `script.js` - Japanese, English, and Traditional Chinese language switching
- `assets/` - SKYBOT logo and supporting visual assets

## Content Scope

The site presents SKYBOT as an investor-facing corporate website. It excludes financial projections, pricing, tokenomics, and detailed financing-related content.
